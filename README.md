# zapret-gui
Простая и минималистичная графическая панель управления (GUI) для популярного обходчика блокировок **Zapret** на Windows. Написана на чистом C++ (WinAPI) без использования тяжелых фреймворков.

## 📌 Возможности
- 🗂 Автоматический поиск файлов конфигурации `general*.bat` в папке программы.
- 📌 Возможность «закрепить» выбранный метод обхода по умолчанию.
- 🔄 Сворачивание панели в системный трей (меню по правому клику мыши).
- 💻 Поддержка автозагрузки вместе с Windows (с автоматическим подхватом рабочей папки).
- 🛡 Быстрый доступ к настройке системной службы (`service.bat`) с запросом прав администратора (UAC).

## 🚀 Установка и использование
1. Перейдите в раздел **Releases** и скачайте `zapret_gui.exe`.
2. Поместите скачанный файл прямо в корень папки вашего **Zapret** (туда, где лежат ваши `.bat` файлы).
3. Запустите программу.
