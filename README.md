# ⚡ Zapret-GUI v1.2

Минималистичная панель управления (GUI) на **чистом C++ (WinAPI)** для утилит обхода блокировок **Zapret** и **TgWsProxy** (от **flowseal**). Мгновенно запускается, весит несколько килобайт и не расходует ресурсы.

---

## 🚀 Что нового в v1.2

* **Интеграция с TgWsProxy:** Добавлен отдельный блок управления. Выбор исполняемого файла `TgWsProxy.exe` происходит напрямую через «Обзор».
* **Раздельный автозапуск:** Три независимых чекбокса в реестре Windows: для самой панели, для закрепленного `.bat` от Zapret и для `TgWsProxy.exe`.
* **Полная адаптация тем:** Все новые чекбоксы и кнопки корректно перерисовываются при переключении светлой/тёмной темы.

---

## 🔥 Возможности

* **🗂 Поиск скриптов:** Автоматически находит файлы `.bat` в папке Zapret.
* **📌 Фиксация:** Закрепление выбранного метода обхода по умолчанию.
* **🎨 Поддержка тем:** Встроенная тёмная (Dark Mode) и светлая темы.
* **🔄 Работа в фоне:** Сворачивание в системный трей с контекстным меню.
* **🛡 Защита от дубликатов:** Мьютекс блокирует запуск нескольких копий панели.
* **⚙️ Управление службой:** Быстрый запуск `service.bat` с автоматическим запросом прав администратора (UAC).

> ⚠️ **Важно:** Графический интерфейс разработан и протестирован для работы с версиями сборок **Zapret** и **TgWsProxy** от автора **flowseal**.

---

## 💻 Установка и использование

1. Скачайте `zapret_gui.exe` из раздела **Releases** и запустите (можно хранить в любой папке).
2. **Для Zapret:** Нажмите «Обзор...», укажите папку со сборкой, выберите `.bat` скрипт и нажмите **«ЗАПУСТИТЬ МЕТОД ZAPRET»**.
3. **Для TgWsProxy:** Нажмите «Обзор...», выберите файл `TgWsProxy.exe` и нажмите **«ЗАПУСТИТЬ TG WS PROXY»**.
4. Настройте автозапуск компонентов с Windows при помощи соответствующих чекбоксов.
