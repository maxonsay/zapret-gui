# zapret-gui

Простая, быстрая и минималистичная графическая панель управления (GUI) для популярного обходчика блокировок **Zapret** на Windows. 

Написана на чистом **C++ (WinAPI)** без использования тяжелых фреймворков, благодаря чему программа практически не потребляет системные ресурсы и мгновенно запускается.

---

## 🚀 Новое в версии v1.1
* 🎨 **Тёмная тема:** Интерфейс стал более современным и комфортным для глаз.
* 📂 **Выбор папки:** Больше не нужно закидывать программу в корень Zapret — просто укажите путь к нему прямо в интерфейсе.
* 🛡 **Защита от двойного запуска:** Программа не позволит открыть несколько своих копий одновременно, забивая трей.

---

## 📌 Возможности

* 🗂 **Автоматический поиск:** Сканирует выбранную директорию на наличие конфигурационных файлов `general*.bat`.
* 📌 **Фиксация метода:** Возможность «закрепить» выбранный скрипт обхода по умолчанию.
* 🔄 **Работа в фоне:** Удобное сворачивание панели в системный трей с контекстным меню по правому клику.
* 💻 **Автозагрузка:** Поддержка запуска вместе с Windows (с автоматическим подхватом рабочей папки).
* 🛡 **Управление службой:** Быстрый доступ к настройке системной службы (`service.bat`) с автоматическим запросом прав администратора (UAC).

---

## ⚙️ Установка и использование

1. Перейдите в раздел [Releases](https://github.com/maxonsay/zapret-gui/releases/tag/v1.1) и скачайте актуальную версию `zapret_gui.exe`.
2. Запустите программу (ее можно хранить в любом удобном месте на диске).
3. В открывшемся окне укажите путь к корневой папке вашего **Zapret** и выберите нужный метод обхода.
